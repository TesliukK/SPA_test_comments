# SPA_Coments


## Встановлення

1. Скопіювати репозиторій: `git clone https://github.com/TesliukK/SPA_test_comments.git`
2. перейти до папки `cd backend`
3. Інсталювати залежності: `npm install`

## Запуск

Start: `npm start`

## Документація
Swagger
http://localhost:5000/docs
